# Corporate Tower Analyzer

A static web app for analyzing and visualizing Satisfactory Corporate Tower climbing statistics.

## About

This project was entirely vibecoded together. Feel free to use it however you'd like!

## Usage

Simply open `index.html` in a browser or host it on any static site hosting service (GitHub Pages, Netlify, Vercel, etc.).

The app fetches data from the Coffee Cup Studios API and displays:
- Overall statistics and leaderboards
- Daily run counts and progress charts
- Individual player statistics and history
- Active run tracking

## Tech Stack

- Pure HTML/CSS/JavaScript (no framework)
- Fully client-side, no backend required
- Mobile-optimized responsive design
